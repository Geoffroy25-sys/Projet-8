# Projet 8 — RPG Dark Souls 
## Description
 Jeu de rôle inspiré de Dark Souls. Arthur combat des Ogres à travers 3 zones pour atteindre et vaincre le Seigneur des Ogres. Le héros gagne de l'expérience, monte en niveau et gère son inventaire pour survivre.
  ## Structure du projet 
  - fondations.py — Bloc 2 : WORKFLOW & FONDATIONS
  - personnages.py — Bloc 3 : ARCHITECTURE POO
  - zones.py — Bloc 3 : ARCHITECTURE POO
  - combat.py — Bloc 3 : ARCHITECTURE POO 
  - persistance.py — Bloc 4 : PERSISTANCE 
  - exceptions.py — Bloc 5 : Qualité
  - tests/test_jeu.py — Bloc 5 : Qualité
  - interface.py — Bloc 6 : Interface graphique
  - main.py — Bloc 6 : Interface graphique
  ## Lancer le jeu 
  python main.py 
  ## Lancer les tests 
  pytest tests/ -v
